Ajuda a millorar la web :P
Si tens ganes i temps per col·laborar...

Pots contribuir amb codi o idees, o promocionant la web a les diferents xarxes socials! Qualsevol bona pensada és benvinguda! :)

Ah, i a més apareixeràs com a col·laborador de la web! I obtindràs 1000 kudos extra si tens un usuari!!!

La web està feta amb les següents tecnologies en el servidor:

  • Java
  • Spring
  • Thymeleaf

En el cantó client hi ha:

  • HTML
  • CSS
  • jQuery + Ajax

La base de dades és MongoDB (NoSQL, es veu al final de BD!).

Totes aquestes eines són gratuïtes, relativament fàcils d'aprendre si en teniu ganes, i molt potents per fer aquesta mena de coses.

El codi de l'aplicació està obert a tothom al GitHub: https://github.com/llop/porra-joc-eda.
És un típic projecte Maven. Podeu pillar el projecte i proposar així canvis i/o millores :D

En la propia aplicació web podem millorar coses, per exemple:

  • "Internacionalitzar" el texte aprofitant les tecnologies emprades.
  • Moure validacions que no es fan al lloc més indicat + tractament d'excepcions.
  • Currar-se un model de dades més maco que l'actual.

Però també es pot aprofitar la infrastructura existent per fer la versió Mobile de La Porra!

Per exemple, per saber a quant està una aposta de resistència pel jugador X, es fa un GET a /tipus/resistencia/kudos/{kudos}/jugador/{jugador}/rondes/{rondes}. Això torna un JSON amb la resposta.

No sería complicat unir forces per fer una senzilla app, que permeti apostar durant la propia final amb l'IPhone o un mòbil Android.
